Developer Notes — MTM Lookup Logic
Per Prashant's spec · Pseudocode for implementation
FunctionPseudocodeUsed For
get_settlement(commodity, curve_date, prompt_date)row = WHERE commodity=X AND curve_date=Y AND prompt_date=Z
IF not found: interpolate between nearest prompt_dates
RETURN settlement		Futures MTM · QP average calculation
qp_avg_mtm(commodity, curve_date, qp_start, qp_end, qty, dir_sign)dates = business_days(qp_start, qp_end)
avg_px = AVG(get_settlement(commodity, curve_date, d) for d in dates)
return (avg_px − trade_px) × dir_sign × qty
[trade_px=None if fully unpriced]		Unpriced physical MTM · Expected MTM on Unpriced rows
Primary key(curve_date, commodity_code, prompt_date)Store full curve every day for historical MTM reconstruction
Interpolationsettlement = S1 + (S2-S1) × (d-d1)/(d2-d1)
Flag: Interpolated = Y		LME prompts beyond 3M where no direct quote exists
